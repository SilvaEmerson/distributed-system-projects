# MOM(Message Oriented Middleware) with RabbitMQ

## Installing

```shell
npm i
```

## Running

### Using Docker:

```shell
docker-compose up
```

and in another shell section:

```shell
node peer[1-2].js [username]
```

### Don't using Docker

Considering you already have RabbitMQ installed on the machine:

```shell
node peer[1-2].js [username]
```
